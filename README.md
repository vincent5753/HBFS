# HBFS
## 介紹 / Intro
This repo is kinda like a warehouse that I put tweaks to my machines that can improve productivity for me.  
這個 repo 對我來說有點像是個放一些我對常用機器的一些調整，整體來說都是些我認為可以增進生產力的設置.
